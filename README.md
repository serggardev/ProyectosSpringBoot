# ProyectosSpringBoot
Primeros proyectos de Spring Boot

Repositorio único para mis proyectos de aprendizaje con Spring Boot y tecnologías relacionadas.

## Estructura

| Proyecto | Descripción | Tecnologías |
|----------|-------------|-------------|
| [taskmanager](/taskmanager/) | API REST para gestión de usuarios y tareas | Java 17, Spring Boot, JPA, H2 |
| frontend-basic | Frontend básico para consumir la API | HTML, CSS, JavaScript |
| android-app | App Android que consume la API | Kotlin, Android SDK |

## Cómo ejecutar cada proyecto

Ver el README específico de cada carpeta.

## Objetivo

Practicar arquitectura completa: Backend → Frontend → Mobile.
